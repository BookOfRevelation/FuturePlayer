# PurePlayer
PurePlayer is a simple mp3 player, developped with Qt5 and based on Microsoft® Windows98 graphical user interface.

![PurePlayer](http://outstream.fr/fp.PNG)

This software has been developped by the team Sydney  Mixtape.

## Features

  - Play mp3, wav and wma files
  - Support drag and drop mechanism
  - Customize playlist system
  - Bind tracks with album cover
  - Beautiful, nostaglic and cosy user interface
  
## Installation

Open the pro file in QtCreator and simply compile and run it.
You can also download a windows installer here : http://outstream.fr/FuturePlayer.msi
